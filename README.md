# Henderson Country Club Website

Live site: https://jonfwilleford.github.io/henderson-country-club/

---

Overview
This is a responsive website built for Henderson Country Club.  
It includes pages for booking, membership, pricing, events, gallery, and restaurant information.

---

Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)
- GitHub Pages (hosting)

---

Project Structure
/index.html → Home page  
/pages/ → All secondary pages  
/css/ → Stylesheets  
/js/ → JavaScript files  
/imgs/ → Images

---

Features
- Responsive mobile-first design
- Navigation with active page highlighting
- Booking form (Formspree integration)
- Multi-page layout
- Image-based hero sections

---

Deployment
This site is deployed using GitHub Pages.

To update:
1. Make changes locally
2. Commit changes
3. Push to `main`
4. Site updates automatically

---

Notes
- Relative paths are used for navigation between pages
- Designed as a static front-end project
